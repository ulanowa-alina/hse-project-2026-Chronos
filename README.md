# hse-project-2026-Chronos

Chronos - десктоп-приложение для управления временем и личной продуктивностью. Работа с системой начинается с персонального дашборда, который служит единой точкой входа и позволяет оценить общую загруженность. Здесь пользователь видит все свои тематические доски и получает автоматическую сводку из трех самых приоритетных дедлайнов, собранных со всех досок.

При переходе к конкретной доске открывается структурированное рабочее пространство с задачами. Все задачи распределяются по трем основным состояниям: планируемые дела, задачи в процессе выполнения и уже завершенные цели. 

На доску можно добавить детальную карточку задачи, которая содержит всю необходимую информацию. Каждая карточка имеет название, описание, таймер до дедлайна и систему цветовых приоритетов (цветовая гамма от красного до синего – оценка срочности).
